Fast Protection Demonstration System Connections

1. General

  • (Optional) USB serial and programming cables can be connected to each of the ZC706 modules for viewing console messages and for Vivado programming or ChipScope diagnosis.  The demonstration system can operate with any or none of these connections present.

2. Event Generator

  • Connect the GPS-PMOD module to ZC706 PMOD1 (J58) pins 1 through 11.  Pin one of the cable is marked as is pin one of the ZC706 header.  Give the GPS-PMOD module as clear a view of the sky as possible and make sure it does not short to the tray – a piece of paper as an insulator is a good idea.
  • Connect a fiber between the event generator on-board SFP module and the on-board SFP module of the mitigation node.
  • Power up the event generator as soon as possible since the GPS-PMOD can take 15 minutes or more to obtain a lock on the satellite signals.  While attempting to achieve lock the LED on the GPS-PMOD module and the ZC706 LED_CENTER (DS9) will blink on and off at 0.5 Hz.  When the GPS-PMOD has achieved lock the ZC706 LED_0 (DS35) will flash once per second.

3. Mitigation Node

  • Mount FMC-4SFP+ module with two SFP modules loaded on FMC-HPC (J37) connector.  Ensure that all switches on the FMC-4SFP+ module are in the '0' position.
  • Mount XM105 module on ZC706 FMC-LPC (J5) connector.
  • Connect fiber from event generator module to on-board SFP module.
  • Connect fiber from rightmost FMC-4SFP+ SFP module to acquisition link 0, node 0.
  • Connect fiber from seecond rightmost FMC-4SFP+ SFP module to acquisition link 1, node 0.
  • Connect Ethernet cable to IOC or to a network switch to which the IOC is connected.

4. Acquisition Link 0, Node 0

  • Mount XM105 module on ZC706 FMC-HPC (J37) connector.  Don't bolt down – the XM105 will get moved around to demonstrate inputs from various acquisition nodes.
  • Connect fiber from mitigation node to on-board SFP module.

5. Acquisition Link 1, Node 0

  • Mount remaining FMC-4SFP+ module on FMC-LPC (J5) connector.
  • Connect fiber from mitigation node to on-board SFP module.
  • Connect fiber from FMC-4SFP+ module SFP to acquisition node link 1, node 1.

6. Acquisition Link 1, Node 1

  • Connect fiber from acquisition node link 1, node 0 to on-board SFP module.
